инструкция по эксплуатации девайса re.сonnect

начало эксплуатации:

1. нажмите на кнопку “пуск” рядом с сенсорным экраном на боковой поверхности устройства
2. отсканируйте qr-код на экране
3. пройдите регистацию/идентификацию на сайте

для сбора данных с вашего головного мозга:

4. на сенсорной панели выберите опцию “загрузка данных”
5. наденьте девайс на свою голову
6. нажмите на кнопку “пуск”
7. ожидайте окончания(в среднем занимает 7 минут)
8. снимите девайс

для создания навыка:

9. на сенсорной панели выберите опцию “создать навык”
10. при выборе следуйте инструкциям на панели
11. нажмите на кнопку “пуск”
12. ожидайте окончания создания(шкала будет высвечиваться на панели)

для выгрузки данных с девайса:

13. на сенсорной панели выберите опцию “выгрузка данных”
14. наденьте девайс на свою голову
15. нажмите на кнопку “пуск”
16. ожидайте окончания(в реднем занимает 5 минут)
17. снимите девайс
